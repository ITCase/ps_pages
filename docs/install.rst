Installing
==========

github
------

.. code-block:: bash

    pip install git+http://github.com/uralbash/pyramid_pages.git

PyPi
----

.. code-block:: bash

    pip install pyramid_pages

source
------

.. code-block:: bash

    git clone git+http://github.com/uralbash/pyramid_pages.git
    cd pyramid_pages
    pip install -e .
